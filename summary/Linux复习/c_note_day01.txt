【1】打开一个终端
     crtl+alt+t

     必须在已经开启一个终端的前提下才有效
     crtl+shift+n  打开的终端和之前的终端在同一个路径下，单独占用一个窗口
     crtl+shift+t  打开的终端和之前的终端在同一个路径下，和之前的终端共用一个窗口


【2】终端字体调整
     crtl+(+)  调大
     crtl+(-)  调小
  
【3】linux@ubuntu：~$
     
     linux:代表当前用户
     ubuntu：代表主机名
     ~:代表当前所处的目录
       ~代表的是当前用户的家目录  ===> /home/linux
     $:权限提示符
       $:普通用户权限，一般工作在此权限下
       #：管理员权限
       从普通用户切换到管理员权限：sudo su 
       从管理员权限切换到普通用户权限:exit
               
       如果在普通用户权限下，exit可以退出终端
       在命令前使用加上sudo表示该命令临时使用管理员权限

【5】清屏操作

【6】基本shell命令
     1---查看目录  ls
         ls后面可以跟路径，表示查看指定路径下的文件
         ls  .. ： 表示查看上一级目录下的文件
         ls /   ： 表示查看根目录下的文件
         ls -l ：以列表的形式显示当前文件夹下的内容
         ls -a ：罗列出当前目录下的所有内容，包括隐藏文件（隐藏文件都是以"."开头）
         ls -la : 以列表的形式查看当前文件夹下的所有的内容，包括隐藏文件
	
	ls -R:
         -rwxrw-r--  1 linux linux      105 Jul 17 14:51 1.txt
      
          第一个符号代表的是当前文件的文件类型
          	linux中一切皆文件，共有7种文件
                    b             c                 d             -            l            s           p
               块设备文件   字符设备文件       目录（文件夹）  普通文件    链接文件     套接字文件   管道文件
          rwxrw-r--:三组可读可写可执行权限
                    第一组代表当前用户对这个文件的权限 rwx -> 可读可写可执行
                    第二组代表所属组对这个文件的权限   rw- -> 可读可写不可执行
                    第三组代表其他普通对这个文件的权限 r-- -> 可读不可写不可执行
          1:代表链接数
          linux：所属用户
          linux：所属组
          105:文件的大写，以字节为单位
          Jul 17 14:51 ：最后修改的时间
          1.txt :文件名
                     
     2---目录切换  cd
         进入到根目录： cd  /
         进入家目录：   cd ~   
         进入到家目录： cd  
         回到上一级目录： cd ..
         ..代表上一级目录    .代表当前目录
         回到上一次工作的目录： cd -
         

         路径：
		绝对路径：从根目录开始，当前文件所在的位置	
                相对路径：从当前目录到想要到的位置(. 和 ..)

                比如：cd /home/linux      绝对路径
                      cd ../../home/linux 相对路径

                      cd  /home/linux/work  绝对路径
                      cd ./work             相对路径
         
     3---目录的创建和删除
         创建一个目录：mkdir 目录名
         创建嵌套目录：mkdir 目录名/目录名/目录名  -p
         删除一个空目录：rmdir 目录名   注意：只能删除空目录


     4---普通文件的操作
          
	1）touch命令
           	touch  1.txt
           创建一个普通文件
           如果这个普通文件已经存在则帮我们更新这个文件的时间戳为系统当前时间
        
        2）echo命令
           echo "farsight"   将farsight输出到终端
           echo "你好" > 1.txt  将"你好"写入到1.txt文件中   会将1.txt中之前的内容给覆盖
           echo "他好" >> 1.txt 将"他好"追加到1.txt文件中，之前的内容保留

	3）cat命令
           查看普通文件中的内容    

        4）rm命令  
           rm 2.txt 将普通文件2.txt删除

           rm 目录名 -r  强制删除目录 


     5---复制和移动
    
         cp  可以复制文件
         cp  普通文件  路径   : 将一个文件复制到指定路径下
         cp  普通文件  路径/文件名  ：将一个文件复制到指定路径下，并命名为指定的文件名
         如果要复制的文件是目录，则需要在命令后面加上-r选项


         mv  可以移动文件
         mv   文件名   路径  ：将一个文件移动到指定路径下
         mv   文件名   文件名 ：对文件进行重命名
 

        
     6---共享文件夹
         设置共享文件夹   vmware -> 虚拟机 -> 设置 -> 选项  -> 共享文件夹   - > 选中总是启动共享文件夹  -> 添加一个共享文件夹  设置主机路径（就是windows上的共享文件夹的路径）
         ubuntu中共享文件夹的路径为:/mnt/hgfs/共享文件夹名           
        
      


【7】vi编辑器
     vi是Linux中最常用的文本编辑器，嵌入式系统开发人员要能熟练使用这个工具，vim是其改进版本    

     vi编辑器共有三种模式：命令行模式，插入模式，底行模式


     1---命令行模式
         进入vi的初始模式，可以通过一些简单的命令对这个文档进行一些操作，比如说复制，剪切，粘贴等
          yy:复制光标所在行
          3yy:复制光标及以下共三行


          dd：剪切光标所在行
          4dd：剪切光标及一下共四行
	d^
	d$	


          p：粘贴到光标的下一行
          P：粘贴到光标的上一行

          u：撤销
          crtl+r：恢复

          gg：将光标定位到第一行
          G： 将光标定位到最后一行


      2---插入模式
          在这种模式下可以进行文档的编写   
          
          从命令行模式下切换到插入模式：
          	i：从当前光标前进入插入模式
                I：从当前光标所在行行首的插入模式
                a：从当前光标的后面进入插入模式
                A：从当前光标所在行的行尾进入插入模式
                o：从当前光标的下一行进入插入模式
                O：从当前光标的上一行进入插入模式
          
          从插入模式进入命令行模式：esc


       
      3---底行模式
          在这个模式下可以进行文件的保存退出等动作
           
          在命令行模式下输入":"进入底行模式
   
          wq：保存退出
          q!：强制退出（不保存退出）
          q：退出
          x：保存退出
          w：保存
          w! 文件名：将文本类型保存到指定的文件中 
          


          vsp 文件名：（竖屏）分屏查看指定文件
          sp  文件名：（横屏）
          wqa：所有文件保存退出
	：% s /new/old
