【1】printf函数
     格式：printf(“格式控制串”，输出表)
     功能：按指定格式向终端输出数据
     返值：正常，返回输出字节数；出错，返回EOF(-1)

     输出表：要输出的数据（可以没有，多个时以“,”分隔）
     格式控制串：包含两种信息
		格式说明：     %[修饰符]格式字符  ，用于指定输出格式
		普通字符：原样输出

		格式字符
			d,i     十进制整数
                        x,X     十六进制无符号整数
                         o        八进制无符号整数
                         u       无符号十进制整数
                         c       输出字符
                         s       输出字符串
                         f       输出浮点数
                         e       按照指数形式输出
                         %%      输出%


		格式字符修饰符
			m（整型数字）       指定输出的数据的域宽，如果数据本身的宽度小于域宽，则左补空格，否则原样输出
                      
 		        .n(整型数字)      对实数,指定小数点后位数(四舍五入)  
					   对字符串,指定实际输出位数   
                             
                        -                  输出数据在域内左对齐（缺省右对齐)
                        +                  指定在有符号数的正数前显示正号(+)

                        0                  输出数值时指定左面不使用的空位置自动填0

                        #                  在八进制和十六进制数前显示前导0，0x

                                           在d,o,x,u前，指定输出精度为long型
                         l                 在e,f,g前，指定输出精度为double型



【2】scanf函数
     格式: scanf(“格式控制串”，地址表）
     功能：按指定格式从键盘读入数据，存入地址表指定存储单元中,并按回车键结束
     返值：正常，返回输入数据个数


     输入分隔符的指定一般以空格、TAB或回车键作为分隔符
     其它字符做分隔符：格式串中两个格式符间字符

     垃圾字符的回收：
	1）加空格,输入的空格、回车、tab键都不作为有效字符被读取
           scanf("%c %c", &c1, &c2);

        2）getchar()一个getchar（）函数可以吸收一个垃圾字符

        3）%*c  读取到的字符不赋值给任何变量


【3】字符串输出函数
     puts()
     格式：puts(字符数组)
     功能：向显示器输出字符串（输出完，换行）
     说明：字符数组必须以‘\0’结束

 
【4】字符串输入函数
     gets()
     字符串输入函数gets格式：gets(字符数组)
     功能：从键盘输入一以回车结束的字符串放入字符数组中，并自动加‘\0’
     说明：输入串长度应小于字符数组维数

       3）while循环语句
          while(循环条件){
		//循环体;
		}

          判断循环条件，如果为真，则执行循环体，执行完循环体后，再次判断循环条件,如果为真，则再次执行循环体.....如果为假，则结束while循环

          练习：猴子吃桃问题：猴子第一天摘下若干个桃子，当即吃了一半，还不瘾，又多吃了一个，
				第二天早上又将剩下的桃子吃掉一半，又多吃了一个。以后每天早上都
				吃了前一天剩下的一半零一个。到第10天早上想再吃时，见只剩下一个
				桃子了。求第一天共摘了多少。

          do{
              //循环体
	      }while(循环条件);

           while与do...while 的区别:while第一次执行的时候，先判断循环条件，再去决定是否执行循环体，
                                    do...while第一次执行的时候，先执行循环体，然后再去判断循环条件
                                    while可能一次都不执行，do...while至少会执行一次



      
       4）for语句

          for(表达式1;表达式2;表达式3){
		循环体
		}

          练习：        *：打印100-1000以内的所有的水仙花数
				水仙花数：是指一个三位数的每一位的立方和还等于这个数，则这个数称之为水仙花数
				num = 个位数的立方 + 十位数的立方 + 百位数的立方
			**：求1000以内的所有完数
				完数：完美的数，一个数的所有因子，出去它本身相加还等于这个数，则这个数称之为完数
					6 = 1 +２＋３
			**：求1000以内所有的质数
				质数：只能够1和它本身整除
			*：打印99乘法表
				1 x 1 = 1
				2 x 1 = 2  2 x 2 = 4
				3 x 1 = 3  3 x 2 = 6  3 x 3 = 9
				...
			**：使用嵌套循环产生下列图案，要求用for嵌套循环
				F
				_FE
				__FED
				___FEDC
				____FEDCB
				_____FEDCBA
